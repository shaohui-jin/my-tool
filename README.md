# my-tool

## 运行环境

### turbo 环境 

```shell
npm install -g turbo
```

### vue-tsc 环境

```shell
npm install -g vue-tsc
```

## 部分特殊处理

### 如果 project 的 utils 中没有 dist

> 在 project 的 utils 目录下执行 `pnpm run build:tsc`

### 如果 project 的 components 中没有 dist

> 在 project 的 components 目录下执行 `pnpm run build:tsc`
